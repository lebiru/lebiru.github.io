Primer on Booting a PC

PCs are versatile machines capable of booting various operating systems. Think of operating systems like the foundation upon which applications are built. Unlike regular apps, operating systems interact closely with the hardware to manage resources and run programs.

While Windows and macOS dominate the market, there are plenty of other operating systems available. Among these alternatives is Linux, an open-source powerhouse with a thriving ecosystem of distributions.

Enter Linux OS

Linux was created in 1991 by Linus Torvalds as a free and open-source alternative to proprietary operating systems. What started as a small project grew into a global phenomenon, with developers and enthusiasts contributing to its success.

Today, there are over 600 Linux distributions, each tailored to specific needs and preferences. Linux is celebrated for its open-source nature, robust security, and unparalleled customization. Whether you're a beginner or an advanced user, there's a Linux distribution for you.

Ubuntu Installation Guide

1. Download the ISO File

  1. Visit the official Ubuntu website.
  2. Select the version of Ubuntu you want to install (e.g., Ubuntu Desktop for general use).
  3. Choose the architecture suitable for your computer (e.g., 64-bit for most modern PCs).
  4. Click the download button and save the ISO file to an accessible folder on your current computer.

2. Create a Bootable USB Drive using Etcher

  1. Download and install Balena Etcher on your current computer.
  2. Insert a USB flash drive (at least 4GB or larger) into your computer.
  3. Open Etcher and:
    1. Select the Ubuntu ISO file you downloaded.
    2. Choose your USB drive as the target device.
    3. Click "Flash" to start creating the bootable USB.
  4. Once completed, safely eject the USB drive.

3. Boot From the USB Drive

  1. Restart your computer and access the Boot Menu. Common keys to access it are F12, Esc, F2, or Delete (this varies by manufacturer).
  2. In the Boot Menu, select your USB drive as the boot device.
  3. Save the changes (if needed) and exit the BIOS or Boot Menu. Your computer will boot into the Ubuntu USB.

4. Explore the Live Environment

  1. When the Ubuntu installer loads, you‚Äôll have the option to "Try Ubuntu" without installing it.
  2. Select "Try Ubuntu" to explore the live environment.
  3. Test hardware compatibility (e.g., Wi-Fi, audio, and display) to ensure everything works properly before installing.

5. Start the Installation

  1. When you're ready, double-click the "Install Ubuntu" icon on the desktop.
  2. Select your preferred language and click "Continue."

6. Configure Settings

  1. Keyboard Layout:
    • Choose your keyboard layout or let Ubuntu detect it automatically.
    • Test typing in the provided box to ensure it's correct.
  2. Updates and Other Software:
    • Select whether to install third-party software (e.g., for graphics and Wi-Fi hardware).
    • Choose "Normal Installation" (recommended) or "Minimal Installation" for a lightweight setup.
  3. Installation Type:
    • For beginners, choose "Erase disk and install Ubuntu" (this will delete all existing data on the drive).
    • Advanced users can manually partition the drive.
  4. Location: Select your time zone by clicking on the map or typing your city.
  5. User Account:
    • Enter your name, choose a username, and set a strong password.
    • Decide if you want Ubuntu to log in automatically or require a password.

7. Install and Finalize

  1. Review your selections and click "Install Now."
  2. Wait for the installation process to complete. This may take 10-30 minutes depending on your system.
  3. Once done, remove the USB drive when prompted and click "Restart Now."
  4. After rebooting, your computer will load into Ubuntu.

8. Post-Installation Setup

  1. Update the System: 
    sudo apt update && sudo apt upgrade -y
  2. Install Additional Software:
    • Use the Ubuntu Software Center to browse and install applications.
    • Alternatively, use the terminal for package installation (e.g., sudo apt install <package-name>).
  3. Configure Settings: Set up your preferred system settings (e.g., display resolution, power management, and keyboard shortcuts).
  4. Explore: Familiarize yourself with the Ubuntu desktop environment, and enjoy your new Linux system!

Linux Mint Installation Guide

1. Download the ISO File

  1. Visit the official Linux Mint website.
  2. Select the edition of Linux Mint that suits your preferences:
    • Cinnamon ‚Äì A modern and feature-rich desktop environment.
    • MATE ‚Äì A lightweight and traditional desktop environment.
    • Xfce ‚Äì The most lightweight option, ideal for older hardware.
  3. Choose the appropriate architecture for your system (e.g., 64-bit for most modern PCs).
  4. Click the download button and save the ISO file to an easily accessible folder on your current computer.

2. Create a Bootable USB Drive using Etcher

  1. Download and install Balena Etcher.
  2. Insert a USB flash drive with at least 4GB of free space into your computer.
  3. Open Etcher and:
    1. Click "Select Image" and choose the Linux Mint ISO file you downloaded.
    2. Select your USB flash drive as the target device.
    3. Click "Flash" to create the bootable USB drive.
  4. Wait for the flashing process to complete, then safely eject the USB drive.

3. Boot From the USB Drive

  1. Restart your computer and access the Boot Menu. Common keys for this include F12, Esc, F2, or Delete (depending on your system).
  2. Select your USB drive from the Boot Menu and press Enter.
  3. Your computer will boot into the Linux Mint installation environment.

4. Explore the Live Environment

  1. Choose the "Start Linux Mint" option from the boot menu.
  2. Linux Mint will load a live environment that allows you to try the system without installing it.
  3. Test your hardware (e.g., Wi-Fi, sound, and display) to ensure everything works as expected.

5. Start the Installation

  1. Double-click the "Install Linux Mint" icon on the desktop to start the installation process.
  2. Select your preferred language and click "Continue."

6. Configure Settings

  1. Keyboard Layout:
    • Select your keyboard layout or let Linux Mint detect it automatically.
    • Type in the test field to ensure the layout is correct.
  2. Updates and Other Software:
    • Choose whether to install third-party software (e.g., for graphics drivers and multimedia codecs).
    • Select "Normal Installation" for a full-featured setup or "Minimal Installation" for a lightweight environment.
  3. Installation Type:
    • Select "Erase disk and install Linux Mint" for a clean installation (this will delete all existing data).
    • Advanced users can select "Something else" to manually partition their disk.
  4. Location:
    • Select your time zone by clicking on the map or typing your city name.
  5. User Account:
    • Enter your name, choose a username, and set a strong password.
    • Decide whether to log in automatically or require a password on boot.

7. Install and Finalize

  1. Click "Install Now" to begin the installation process.
  2. Linux Mint will copy files and install the operating system, which may take 10-30 minutes.
  3. When prompted, remove the USB drive and click "Restart Now."
  4. After rebooting, your system will load into the new Linux Mint installation.

8. Post-Installation Setup

  1. Update the System: 
    sudo apt update && sudo apt upgrade -y
  2. Install Additional Software:
    • Use the Software Manager to install applications you need.
    • Alternatively, use the terminal for package installation (e.g., sudo apt install <package-name>).
  3. Configure Settings:
    • Customize your desktop environment, including themes, icons, and wallpaper.
    • Set up power management and other system preferences.
  4. Explore: Familiarize yourself with Linux Mint and enjoy your new system!

  2. The live environment will load, allowing you to try Fedora without installing it.

5. Start the Installation

  1. Double-click the "Install to Hard Drive" icon on the desktop.
  2. Select your preferred language and click "Continue."

  1. Keyboard Layout:
    • Choose your keyboard layout or let Fedora detect it automatically.
    • Test by typing in the provided test field to ensure it works correctly.
  2. Installation Destination:
    • Select the disk where Fedora should be installed.
    • Choose "Automatic Partitioning" for beginners or "Custom Partitioning" for advanced users.
  3. Time and Date:
    • Set your time zone by clicking on the map or selecting from the dropdown menu.
  4. User Account:
    • Create a root password and a regular user account with a strong password.
    • Check the option to make the user an administrator, if needed.

7. Install and Finalize

  1. Click "Begin Installation" to start the process.
  2. Fedora will copy files and configure the system. This may take 15-30 minutes.
  3. Once the installation is complete, remove the USB drive and click "Reboot" to restart your computer.
  4. Fedora will load for the first time after rebooting.

8. Post-Installation Setup

  1. Update the System: 
    sudo dnf update -y
  2. Install Additional Software:
    • Use the "Software" application to browse and install applications.
    • Alternatively, use the terminal for package installation: 
      sudo dnf install <package-name>
  3. Configure System Settings:
    • Set up display preferences, power management, and other system options.
  4. Explore Fedora:
    • Get familiar with the GNOME desktop environment and its features.
